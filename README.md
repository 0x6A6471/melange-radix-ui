# melange-radix-ui

[Melange](https://melange.re) bindings for [Radix Primitives](https://www.radix-ui.com/primitives).


## Installation

add the following to your `dune-project` file:

```lisp
(pin
 (url "../melange-radix-ui")
 (package (name melange-radix-ui)))
```

```sh
dune pkg lock
dune build
```
